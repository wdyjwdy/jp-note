# N4

## ～より～

- 「名词」＋「より」：表示比较。否定句中使用[「～ほど～ません」](#ほどません)。
- 「疑问词」＋「より」：表示最高级。

```js
1. 私は犬より猫が好きです。 // 相比狗，我更喜欢猫。
2. 私は何より音楽が好きです。 // 我最喜欢音乐。
```

## ～ほど～ません

- 「名词」＋「ほど」：表示比较。肯定句中使用[「～より～」](#より)。

```js
今は昔ほど楽しくないです。 // 现在没有以前快乐。
```

## ～より～のほう

- 「名词」＋「より」＋「名词」＋「のほう」：两件事物进行对比。

```js
日本語より英語のほうが難しい。 // 英语比日语难。
```

## ～と～とどちら

- 「名词」＋「と」＋「名词」＋「とどちら」：两件事物进行对比的提问。通常使用「のほう・どちらも」回答。

```js
1. コーヒーと牛乳とどちらがいいですか。 // 喝咖啡还是牛奶？
   牛乳のほうがいいです。 // 喝牛奶。
2. 東京まで新幹線と電車とどちらが速いですか。 // 到东京新干线快还是电车快？
   新幹線のほうが速いです。 // 新干线更快。
   どちらもあまり速くありません。 // 都不怎么快。
```

## ～ながら～

- 「ます形」＋「ながら」：表示动作同时进行。后者为主要动作。

```js
音楽を聞きながら勉強します。 // 边听歌边学习。
```

## ～ところです

- 「字典形」＋「ところ」：即将做。
- 「ている」＋「ところ」：正在做。
- 「た形」＋「ところ」：刚刚做完。

```js
1. 試合が始まるところです。 // 比赛就要开始了。
2. 牛乳を飲んでいるところです。 // 正在喝牛奶。
3. ご飯を食べたところです。 // 刚吃完饭。
```

## ～まで～

- 「名词・字典形」＋「まで」：某时刻前的持续性动作。
- 「名词・字典形」＋「までに」：某时刻前的瞬间动作。

```js
1. 十二時まで勉強します。 // 学到十二点。
2. 九時までに寝ます。 // 九点前睡觉。
```

## ～ませんか

- 「ます形」＋「ませんか」：表示提议。

```js
1. 一緒に食べませんか。 // 一起吃饭吗？
2. うちに来ませんか。 // 来我家吗？
```

## ～ましょうか

- 「ます形」＋「ましょうか」：表示提议。使用「ませんか」要更加委婉。
- 「ます形」＋「ましょう」：默认提议已经通过。
- 「よう形」＋「か」：比「ましょうか」更亲近。

```js
1. 帰りましょうか // 我们回家吧。
2. 何を食べましょうか。 // 吃什么？
3. 行きましょう。 // 我们走吧。
4. 持ちましょうか。 // 我帮你拿吧。
5. 持とうか。 // 我帮你拿吧。
```

## ～られます

- 「可能形」：能够做某事。

```js
1. 日本語が話せます。 // 会说日语。
2. 水は飲めません。 // 水不能喝。
```

## ～ができます

- 「名词」＋「ができます」：能够做某事。比[「～られます」](#られます)更正式。
- 「字典形」＋「ことができます」

```js
1. 日本語ができます。 // 会说日语。
2. 水は飲むことができません。 // 水不能喝。
```

## ～たことがあります

- 「た形」＋「ことがあります」：以前做过某事。

```js
1. 日本へ行ったことがあります。 // 去过日本。
2. 一度も海を見たことがない。 // 一次海都没看过。
```

## ～ことがあります

- 「辞書形・ない形＋ことがあります」：偶尔发生某事。
- 「辞書形・ない形＋こともあります」

```js
1. 時々朝ご飯を食べことがあります。 // 偶尔会吃早饭。
2. 晩御飯を食べないこともあります。 // 不吃晚饭的时候也有。
```

## ～てもいい

- 「て形・い形くて・な形で・名词で」＋「もいい」：表示让步。
- 「ない形・い形く・な形で・名词で」＋「なくてもいい」

```js
1. 狭くてもいいです。 // 小一点也可以。
2. 近なくてもいいです。 // 不那么近也可以。
```

## ～てはいけません

- 「て形・い形くて・な形で・名词で」＋「はいけません」：表示禁止。

```js
撮っては行けません。 // 不能拍照。
```

## ～なければなりません

- 「ない形・い形く・な形で・名词で」＋「なければなりません」：表示必须。

```js
朝ご飯を食べなければなりません。 // 必须吃早饭。
```

## ～がほしいです

- 「名词」＋「がほしいです」：表示希望。第三者的希望用[「～がります」](#がります)表示。

```js
車が欲しいです。 // 想要一辆车。
```

## ～たいです

- 「ます形」＋「たいです」：表示希望。第三者的希望用[「～がります」](#がります)表示。

```js
海を見たいです。 // 想看海。
```

## ～がります

- 「名词」＋「をほしがります」：表示第三者的希望。
- 「ます形」＋「たがります」：表示第三者的希望。

```js
1. 犬は水を欲しがります。 // 狗想喝水。
2. 誰でも行きたがりません。 // 谁都不想去。
```

## ～といいです

- 「普通形」＋「といいです」：表示希望发生的情况。句尾常加「ね・なあ・けど」。

```js
1. 仕事を見つかるといいですね。 // 能找到工作就好了。
2. 明日は雨が降らないといいなあ。 // 明天不下雨就好了。
```

## ～そうです

- 「ます形・い形・な形」＋「そうです」：表示推断。
- 「ます形」＋「そうもありません」：否定形式。
- 「い形く・な形では・名词では」＋「なさそうです」：否定形式。
- いい　→　よさそうです
- ない　→　なさそうです

```js
1. 美味しいそうです。 // 看起来很好吃。
2. 雨は降りそうもありません。 // 看起来不会下雨。
```

## ～がっています

- 「い形・な形」＋「がっています」：表示第三者的感受。
- 「い形・な形」＋「がります」：陈述一般倾向。

```js
1. 桜は寂しがっています。 // 小樱感到很伤心。
2. 彼は運動を嫌がります。 // 他很讨厌运动。
```

## ～まま～

- 「た形・ない形・い形・な形な・名词の」＋「まま」：表示状态一直持续。通常导致了不好的后果。

```js
1. 座ったまま寝た。 // 坐着睡着了。
2. 靴のまま家に入る。 // 穿着鞋进房间。
```

## ～から～

- 「简体・敬体」＋「から」：表示原因。
- 「简体」＋「からです」：表示原因。

```js
1. 危ないですから、触らないでください。 // 危险，请别碰。
2. 寒いからです。コートを着ています。 // 因为很冷，所以穿着外套。
```

## ～ので～

- 「简体・な形な・名词な」＋「ので」：表示原因。不能接命令句。比[「～から～」](#から)更正式。

```js
頭が痛いので、今日は休みます。 // 头很痛，今天请假。
```

## ～て～

- 「て形・い形くて・な形で・名词で・ない形なくて」：表示原因或理由，通常用于表达情感。不能接劝诱句。

```js
友達がなくて、寂しい。 // 没有朋友，很寂寞。
```

## ～に～

- 「动名词・ます形」＋「に」：表示移动的目的。

```js
散歩に行く。 // 去散步。
東京へ遊びに行く。 // 去东京玩。
```

## ～ために～

- 「动名の・字典形」＋「ために」：表示行为的目的。接在意志动词后。

```js
日本語を勉強するために、日本へ行く。 // 为了学日语，去日本。
```

- 「动名の・字典形」＋「ため」：表示行为的理由。接在意志动词后。

```js
会議に出席するため、日本へ行く。 // 为了开会，去日本。
日本語を勉強するための本。 // 学习日语的书。
```

参考：[「ために・ように」的区别](../vocab/compare.md#ためにように)

## ～ように～

- 「ない形・字典形」＋「ように」：表示期待的状态。接在非意志动词后。

```js
風が入るように、窓を開けた。 // 为了让风进来，打开了窗户。
```

参考：[「ために・ように」的区别](../vocab/compare.md#ためにように)

## ～し～し～

- 「简体」＋「し」：列举多个理由

```js
今日は休みだし、晴れだし、散歩に行こう。 // 今天放假，天气也好，去散步吧。
この牛乳は美味しいし、安いです。 // 这个牛奶又好喝又便宜。
時間もないし、お金もないし、旅行はまた今度にしよう。 // 时间也没有，钱也没有，旅游还是下次吧。
```

## ～たり～たり～

- 「た形」＋「り」：列举多个动作。末尾通常接「する」。

```js
休みの日は家でアニメを見たり、寝たりする。 // 放假的时候，在家看动漫或睡觉。
何でも食べたり飲んだりできます。 // 能吃能喝。
雨が降ったり、止んだりする。 // 雨时下时停。
```
